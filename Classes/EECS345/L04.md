- usefull functions
	- <code>list?</code>
		- returns true if the parameter is a list
	- <code>pair?</code>
		- returns true if the parameter has a car and a cdr
	- ex:
		- <code>(list? '())</code> => true
		- <code>(pair? '())</code> => false
